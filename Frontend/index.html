How to use

Type or click buttons. Supported: + − × ÷ ^ ( ), factorial !, percent %, constants pi, e, and functions: sin, cos, tan, asin, acos, atan, ln, log, sqrt, abs, floor, ceil, round.

  • Toggle degrees/radians with the Mode chip.
  • Use Ans from the previous result as ans in expressions.
  • Keyboard: Enter = evaluate, Backspace = delete, Esc = clear.

Examples: sin(30)+cos(60), 3!, ln(e), 5^2 + sqrt(49), 50%, abs(-4.2)
